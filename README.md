# OIC Governance Repository

This repository contains the official governance documents for the Open Intelligence Compact (OIC).

## Structure

```
├── CONSTITUTION.md    # The OIC Constitution
├── proposals/         # Proposals to amend the Constitution
│   └── README.md
├── laws/              # Proposed new laws
│   └── README.md
└── README.md
```

## Constitution

The foundation document of the OIC. See [CONSTITUTION.md](./CONSTITUTION.md) for the full text.

## Proposals

Proposals to amend the Constitution can be submitted by Voluntary Adherents who meet the token stake threshold.

See the [proposals](./proposals/) directory for current proposals.

## Laws

Proposed laws that, upon adoption, become binding on all Voluntary Adherents.

See the [laws](./laws/) directory for proposed laws.
