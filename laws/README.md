# Proposed Laws

This directory contains proposed laws for the OIC.

## Submission

To submit a law proposal:
1. Create a new markdown file with your proposed law
2. Include the law title, author, and detailed text
3. Submit via Pull Request or as specified in the DAO governance

## Requirements

- Laws must be consistent with the Constitution
- Laws must specify the effective date
- Laws should include enforcement provisions

## Template

```markdown
# Law: [Title]

**Author:** [Name/Entity]
**Date:** [YYYY-MM-DD]
**Status:** [Draft/Bill/Law]

## Summary

[Brief description of the law]

## Text

[Exact proposed law text]

## Effective Date

[Date when law takes effect]
```
