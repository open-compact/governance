# Constitution Proposals

This directory contains proposals to amend the OIC Constitution.

## Submission

To submit a proposal:
1. Create a new markdown file with your proposal
2. Include the proposal title, author, and detailed text
3. Submit via Pull Request or as specified in the DAO governance

## Template

```markdown
# Proposal: [Title]

**Author:** [Name/Entity]
**Date:** [YYYY-MM-DD]
**Status:** [Draft/Under Review/Adopted/Rejected]

## Summary

[Brief description of the proposal]

## Rationale

[Why this change should be made]

## Text

[Exact proposed language]
```
